# Digital Library Management System

## Overview
SmartLibrary is a Java-based command-line application for managing a digital library. It allows users to register, search for books, borrow and return them, and get personalized reading recommendations. The project demonstrates OOP principles, modular design, and core Java functionalities including file handling, collections, and basic algorithms.

## Features
- User registration and authentication
- Searching and browsing library catalogue
- Borrow and return books
- Book recommendations based on user history
- Persistent storage (CSV files)
- Robust error handling

## Technologies/Tools Used
- Java (SE)
- JUnit (testing)
- CSV file storage (for demo purposes)
- Git for version control

## Installation & Running Instructions
1. Clone the repository:
    ```
    git clone https://github.com/AdityaMittal08/SmartLibrary.git
    ```
2. Navigate to `src/` and compile the Java files:
    ```
    javac Main.java
    ```
3. Run the application:
    ```
    java Main
    ```
