# Project Statement

## Problem Statement
Many students and educators lack easy access to a system to manage academic resources like books, keep track of borrowing/returning, and get recommendations. Existing solutions are often over-complicated or lack personalized features.

## Scope
The project builds a simple, modular library management system with book search, user management, borrowing/returning flow, and personalized recommendations.

## Target Users
- Students
- Educators
- Small libraries/book clubs

## High-Level Features
- User registration & management
- Adding/searching/borrowing books
- Recommendation engine for personalized reading suggestions
- Persistent local storage (CSV)
